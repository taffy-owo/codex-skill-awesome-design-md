# getdesign CLI

Install a DESIGN.md template into your project.

## Usage

```bash
npx getdesign add ibm
npx getdesign list
```

## Commands

- `add <brand>`: install selected template to project root as `DESIGN.md`
- `list`: list all available brands with one-line descriptions

## Options

- `--force`: overwrite existing target file
- `--out <path>`: write to a custom path
